# Partial Fraction Trainer
This web app is for students to train their skills on expressing fractional expressions to partial fractions.
Questions are generated with different level of difficulty. For gamification fun, a timer is included to record the total time taken to express a generated fractional expression into its partial fractions correctly.

## A. Setting the Parameters ##
1.  Checked boxes augment the difficulty of the questions by allowing the described case.
2.  Some boxes are checked by default. Users may uncheck them.
3.  For example, checking the improper allowed, generated problem will include improper fraction expressions.
4.  At default, the denominator of the current problem is factorised. Uncheck it to increase the challenge.
5.  Clicking the "new problem" button will apply the parameter settings on the new generated problem.

## B. Training on the Partial Fraction Trainer ##
1.  To start answering a question, click "add step" and follow the prompt.
2.  Click "Check" to verify if the response is correct. Green for correct. Red for error.
3.  Depending on the given problem there will be 3 or 4 steps to complete.
4.  Question ends when correct solution is submitted. 

## C. Gamification ##
1.  The timing feature allows user to time the attempt. 
2.  Click new problem, upon receiving the question to do, click start to begin timing the attempt.
3.  The timer will continue to run until the final correct answer is submitted and verified to be correct.
4.  The time shown is the time taken for the problem. Shorter time with correct answer indicate proficiency.
